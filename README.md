[Minimalist Filament holder-bom.csv](https://github.com/user-attachments/files/29522141/Minimalist.Filament.holder-bom.csv)# FIlament_Spool_Holder
Custom Filament spool holder, capable of holding both a spool and box using the minimal amount of filament possible.
I made this project because my room is very messy. I need a dedicated place to hold my filament. 

I learned how to create CAD with increased parametric capability, along with optimizing 3D prints to minimize the use of filmanet.
